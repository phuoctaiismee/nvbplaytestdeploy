The blogs infomation on the NVBPlay store.

[Demo](https://nvbplay-prismic-frontend.vercel.app/)

## Deployment

[Deployment](https://www.google.com)

## Dev Install

**Please use Yarn instead of Npm to contact modules packages:**

```bash
yarn install
```

or

```bash
yarn
```

**Start cmd**

```bash
yarn start
```

**Build cmd**

```bash
yarn build
```
